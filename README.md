# 📉 Estatística e Probabilidade Prática: Python & R

Este repositório é um guia completo de estudos e biblioteca de consultas práticas voltada para a ciência de dados e estatística. O projeto reúne códigos paralelos desenvolvidos em **Python** e **R** abordando desde a análise exploratória até modelos probabilísticos avançados.

---

## 🗺️ Organização Geral do Projeto

O repositório está estruturado em grandes módulos temáticos:

### 1. 📊 Distribuições Discretas
Modelos onde o espaço amostral é contável.
*   **Modelos incluídos:** Uniforme Discreta, Bernoulli, Binomial, Poisson, Hipergeométrica, Binomial Negativa e Multinomial.
*   *Foco:* Cálculo de PMF (Função de Massa de Probabilidade), CDF acumulada e gráficos de barras/frequência.

### 2. 📈 Distribuições Contínuas (em breve)
Modelos aplicados a variáveis de escala contínua (tempo, peso, altura).
*   **Modelos incluídos:** Uniforme Contínua, Gama, Exponencial, Normal (Gaussiana), Qui-Quadrado, t-Student, F de Snedecor, entre outras.

## 🛠️ Tecnologias e Bibliotecas Utilizadas

### Ecossistema Python 🐍
*   `scipy.stats` (cálculos matemáticos e distribuições)
*   `matplotlib` & `seaborn` (visualizações estatísticas de alto nível)
*   `numpy` & `pandas` (estruturação de vetores e dataframes)

### Ecossistema R 📊
*   `ggplot2` (gráficos geométricos e de densidade)
*   `dplyr` & `tidyr` (manipulação descritiva de dados)
*   Funções base de distribuição (`d-`, `p-`, `q-`, `r-`)
